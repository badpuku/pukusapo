# puku-form

## クイックスタート

1. `npm ci`
2. `npm run dev`
3. http://localhost:3000/

## supabase ローカル開発環境

### 前提条件

- `npm ci`が実行済みであること
- Docker が使用可能であること

### クイックスタート

1. `supabase start`
2. http://127.0.0.1:54323/project/default

※ 初回起動はイメージのビルドから始まるので、時間がかかります
